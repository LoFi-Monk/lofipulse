# Planning Documents

This directory is the dedicated space for storing project plans, design documents, and architectural notes.

## Purpose

- **Drafting**: Use this space to write out ideas and specifications before coding.

## Usage Note for Agents

Documents in this directory are **plans**. They describe systems or features to be implemented.

These documents are not the active project. Do not assume that a design document found here represents the _entire_ workspace or the currently active implementation. The are ideas. Actual implementation can be found outside of this directory in the active workspace.
